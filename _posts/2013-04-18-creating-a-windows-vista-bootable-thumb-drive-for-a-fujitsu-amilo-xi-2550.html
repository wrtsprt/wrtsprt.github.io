This is a short protocol of my experience reinstalling Windows Vista on a Fujitsu AMILO XI 2550.

I basically followed this guide: http://www.raymond.cc/blog/how-to-burn-downloaded-windows-vista-to-dvd/. A couple of small remarks though:

  • alternative Vista files download source: http://www.lolametro.de/de/windows-iso-download-links/
  • after step 5 I used http://www.wintobootic.com to put the ISO on a USB stick
  • the catch: make sure to add the onboard RAID controller driver to the install medium (otherwise there will be no drives visible to install Windows)
    • go to http://support.ts.fujitsu.com/Download/Index.asp
    • serial: ysvb012285
    • select Windows Vista
    • select SATA -> JMicron RAID Controller (Onboard) -> RAID controller JMB36X
